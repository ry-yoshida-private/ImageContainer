# binary_image

## Overview

Utilities for working with binary images, including connected components analysis and connectivity definitions.

## Components

| Component | Description |
|-----------|-------------|
| [container.py](./container.py) | Binary image container class with validation and connected components analysis. |
| [connectivity.py](./connectivity.py) | Defines connectivity types (4-connectivity and 8-connectivity). |
| [connected_components.py](./connected_components.py) | Data class for representing connected components results. |
